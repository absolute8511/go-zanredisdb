# go-zanredisdb

The official Go SDK package for [ZanRedisDB][zan].

### Docs
See [godoc][zan_godoc] and the test case for examples of client usage.

### Tests
Run `go test` (which requires the ZanRedisDB cluster inited with the test namespace).
